# Knowledge

Base de conocimiento de Rubik, organizada por dominio de negocio:

- `materials/`: materiales y sus características.
- `providers/`: conocimiento sobre proveedores.
- `polyboard/`: conocimiento y documentación vinculados con PolyBoard.
- `machines/`: máquinas, capacidades y documentación relacionada.
- `manuals/`: manuales autorizados.
- `processes/`: procesos operativos validados.
- `pricing/`: conocimiento y criterios de precios validados.

Todo contenido deberá contar con procedencia, responsable y validación. No se almacenarán secretos ni datos personales sin una política aprobada.

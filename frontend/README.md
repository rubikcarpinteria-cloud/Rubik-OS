# Frontend

Aplicación web de Rubik OS construida con React, Vite y TypeScript.

## Comandos

- `pnpm dev`: servidor de desarrollo.
- `pnpm build`: typecheck y build de producción.
- `pnpm test`: pruebas con Vitest y Testing Library.
- `pnpm typecheck`: validación estática de tipos.

La pantalla actual es exclusivamente técnica y temporal. Las únicas credenciales permitidas aquí son `VITE_SUPABASE_URL` y `VITE_SUPABASE_ANON_KEY`; nunca debe utilizarse `SUPABASE_SERVICE_ROLE_KEY` en frontend.

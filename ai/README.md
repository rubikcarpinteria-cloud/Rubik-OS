# AI

Capacidades de inteligencia artificial de Rubik OS, organizadas de forma explícita:

- `core/`: componentes compartidos por todas las capacidades de IA.
- `agents/`: agentes especializados por área de Rubik.
- `prompts/`: prompts oficiales, versionados y documentados.
- `memory/`: mecanismos de memoria persistente y contexto.
- `evaluations/`: pruebas y evaluación reproducible de calidad.

Los conectores específicos de proveedores pertenecen a `integrations/`. Este directorio todavía no contiene implementaciones ni reglas de negocio.
